# Catálogo de skins battle royale

## Base de datos

`campus_skins_battle_royale`

## Colección principal

- skins

## Decisiones del modelo

Se utilizó una única colección llamada **skins**, ya que cada documento representa una skin independiente y no comparte información con otras entidades.

El modelo se mantuvo simple porque el ejercicio está enfocado en documentos básicos e inserciones mediante `insertMany()`.
